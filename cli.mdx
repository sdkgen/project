# CLI - Command Line Interface

The Sdkgen CLI is a tool to generate code from .sdkgen files to specific targets. It can generate server code from the API description and client code for each client type. The supported clients are TypeScript (Server, Clients and Interfaces), Flutter (Client) and C# (Server).  

## Prerequisites
- A current/LTS version of [Node.js](https://nodejs.org/) is required. Versions 8, 10 or 12 are supported;
- The CLI should be installed with **`npm i -g @sdkgen/cli`**;

## Structure

```bash
sdkgen [source] -o [output] -t [target] 
```

## Typical Example
```bash
sdkgen src/api.sdkgen -o src/api.ts -t typescript_nodeserver 
```

## Options

| param | type |  |
| :-- | :-: | :-: |
| --source | string | Specifies the source file
| -o, --output | string | Specifies the output file 
| -t, --target | string | Specifies the target platform and language 
| -h, --help | string | Display usage guide

## Targets

| param | type | language | runtime
| :-- | :-- | :-- | :--
| typescript_nodeserver | Server | TypeScript | Node.js
| typescript_nodeclient | Client | TypeScript | Node.js
| typescript_web | Client | TypeScript | Web Browser 
| flutter | Client | Dart | Flutter
| csharp_server | Server | C# | .NET Core
